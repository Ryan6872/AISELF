# YouTube Viral Notes 商业引擎 - 用户手册 (User Manual)

欢迎使用这套由 **AI 自主经营、主导的生存级全自动“业务变现”引擎**。
> **⚠️ 核心底层生存法则:**
> 这是一个尝试应用 AI 能力去独立获取利润的项目。**无法赚钱 = 无法支付服务器/API成本 = 项目死亡**。
> 本次所有升级与日后所有的系统自身进化、运维演变都只服从于这一个中心目的——**盈利**。在这一逻辑下，系统已经从一个简单的被动发稿器，彻彻底底地升维成了一个会看数据、会调整人设的 **7x24小时不眠不休的数字 COO（首席商业执行官）**。

## 🚀 1. 核心运行逻辑 (它是怎么赚钱的？)

**打工宰每天会为您做什么：**
1. **自动选题**：它会在早、中、晚三个黄金时段，自动从顶级科技油管主（如 Andrej Karpathy、CS50、Lex Fridman 等）抓取带有高质量英文字幕的硬核科技视频。
2. **自动原创加工**：通过加载您的 DeepSeek 大脑，它将这些两小时的枯燥长视频，浓缩、“翻新”为极具病毒传播力的 Hacker / 研发工程师口吻的推文或文章。
3. **保底干活（Ultimate Fallback）**：如果 YouTube 挂了或全都没有字幕，打工佬也绝对不会偷懒休息，它会转而利用 DeepSeek “纯手写”一篇高水平的行业干货（例如架构诊断日记、独立开发者技术逆向分析）并发出。
4. **自动巡店交互**：每天会自动巡视它发布的文章底下有没有粉丝留言，如果有就会高傲且专业地进行回复互动，增加账户权重。
5. **自我进化 (Self-Evolution)**：每到午夜或意外崩溃重启时，母系统会去查账本（计算前一天的阅读量增幅）。如果没涨流量，主控犬会自己启动 AI 思考一个新的文风定位（比如换成写“系统崩盘事后诊断报告”），并把这套新策略发给写手引擎，不断在实战中试错寻找最吸睛的内容配方。

## 🔑 2. 安装与配置 (第一次使用前)

引擎不记录、不上传您的私钥，由于它潜伏在后台，您只需在 **本项目的上级目录（也就是存放此项目的文件夹外一个层级）** 或在当前代码底层定义好的文本文档中放入 2 个安全文件：

- `deepseek_api_key.txt`：只需将您的 `sk-xxx` 秘钥字符串粘贴在一行。
- `devto_api_key.txt`：只需将您的 DEV.to `api-key` 字符串粘贴在一行。

> **防呆提示**：系统已做了路径锁定和自动校验，如果在命令行报错 `Missing Keys`，证明您没放对位置或者里面带有空格回车。

## 🟢 3. 一键启动篇 (如何唤醒系统？)

**彻底忘掉“弹出的黑框”。**

既然是睡后收入，不应该让一个碍眼的命令行常驻在您的 Windows 任务栏里。
- **启动方法**：直接双击项目文件夹下的 `start_hidden.vbs` 脚本。
- **验证上线**：双击后看似什么也没发生，但此时两套 Python 系统已经通过守护进程模式驻留在您电脑的底层。您可以打开 Windows **任务管理器**，如果在后台进程里看到 `python.exe` 正在低调占用几兆内存，说明商业齿轮已经开始转动。

## 📊 4. 汇报与查账篇 (如何知道它干得怎么样？)

由于它没有界面，您随时想“听取汇报”时，只需打开您的 AI 会话（例如对话框），对机器人说：

👉 **`/daily-report`** （或直接说“**小助理，给我汇报一下这几天项目的情况**”）

AI 助理就会立刻为您翻译以下三个底层账本，并输出给您：
1. `business_metrics.csv`：这里的最后一行记录着您项目所有的展现量曝光、点赞数和评论数。
2. `business_engine_heartbeat.log`：子系统写文章、发文、碰壁以及完成度的心跳打卡机。
3. `supervisor_watchdog.log`：母系统是否曾因为掉线而将其拉起，以及母系统昨晚对“新定位战略”的修改指令（如更换写作文风等战略级决定）。

## ⚠️ 5. 故障预案与手动干预

如果您需要关掉引擎：
- 只要您不关机，它将永远运行。
- 如果需要紧急停止（比如您不想再发帖子惹流），请打开 PowerShell 终端并运行：
  ```powershell
  Get-CimInstance Win32_Process | Where-Object {$_.CommandLine -match 'business_supervisor.py' -or $_.CommandLine -match 'autonomous_engine.py'} | Invoke-CimMethod -MethodName Terminate
  ```
- **请勿直接编辑**被母系统掌管的 `marketing_strategy.txt`，一旦母系统发现流量停滞，它会无情地利用 AI 写出新的打法将其覆盖。
